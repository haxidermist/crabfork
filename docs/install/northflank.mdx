Deploy Crabfork on Northflank with a one-click template and access it through the web Control UI.
This is the easiest "no terminal on the server" path: Northflank runs the Gateway for you.

## How to get started

1. Click [Deploy Crabfork](https://northflank.com/stacks/deploy-crabfork) to open the template.
2. Create an [account on Northflank](https://app.northflank.com/signup) if you don't already have one.
3. Click **Deploy Crabfork now**.
4. Set the required environment variable: `CRABFORK_GATEWAY_TOKEN` (use a strong random value).
5. Click **Deploy stack** to build and run the Crabfork template.
6. Wait for the deployment to complete, then click **View resources**.
7. Open the Crabfork service.
8. Open the public Crabfork URL at `/crabfork` and connect using the configured shared secret. This template uses `CRABFORK_GATEWAY_TOKEN` by default; if you replace it with password auth, use that password instead.

## What you get

- Hosted Crabfork Gateway + Control UI
- Persistent storage via Northflank Volume (`/data`) so `crabfork.json`,
  per-agent `auth-profiles.json`, channel/provider state, sessions, and
  workspace survive redeploys

## Connect a channel

Use the Control UI at `/crabfork` or run `crabfork onboard` via SSH for channel setup instructions:
